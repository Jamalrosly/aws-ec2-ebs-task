# AWS EC2 + EBS Task

## Task Description

Launch Linux and Windows EC2 instances with web servers.
Create a 5GB EBS volume, attach it to the instance, create a snapshot, and create a new volume from the snapshot.

## Services Used

* Amazon EC2
* Amazon EBS

## Steps Completed

1. Launched Linux EC2 instance and installed Apache web server.
2. Launched Windows EC2 instance and installed IIS web server.
3. Created a 5GB EBS volume.
4. Attached the EBS volume to the Linux EC2 instance.
5. Formatted and mounted the volume.
6. Created a snapshot from the volume.
7. Created a new volume from the snapshot.
